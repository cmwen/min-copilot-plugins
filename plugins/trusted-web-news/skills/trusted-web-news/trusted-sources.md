# Trusted default sources

## AI

- OpenAI News — https://openai.com/news/rss.xml
- AWS Machine Learning Blog — https://aws.amazon.com/blogs/machine-learning/feed/
- GitHub Blog (AI/ML) — https://github.blog/feed/
- Hugging Face Blog — https://huggingface.co/blog/feed.xml
- Google AI Blog — https://blog.google/technology/ai/rss/

## Web

- Cloudflare Blog — https://blog.cloudflare.com/rss/
- WebKit Blog — https://webkit.org/blog/feed/
- Mozilla Hacks — https://hacks.mozilla.org/feed/
- W3C News — https://www.w3.org/blog/news/feed/
- web.dev — https://web.dev/feed.xml
- Chromium Blog — https://blog.chromium.org/feeds/posts/default?alt=rss
- Microsoft Edge Blog — https://blogs.windows.com/msedgedev/feed/
- Node.js Blog — https://nodejs.org/en/feed/blog.xml

## Selection rules

- Prefer the source that first publishes the news.
- Use vendor or project blogs before third-party media when both cover the same announcement.
- Exclude sources that do not expose a clear publication date.
